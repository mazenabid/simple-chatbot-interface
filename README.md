# ChatGPT UI Clone 💬

A simple ChatGPT-style chat interface built with vanilla HTML, CSS, and JavaScript.  
Messages render in real-time with a clean, modern chat layout and a fixed input bar.

---

## ✨ Features

- ChatGPT-style UI layout
- Real-time message rendering
- Clean message bubbles
- Sticky input area
- Enter-to-send support
- Fully responsive layout

---

## 🧠 Built With

- HTML
- CSS
- JavaScript (Vanilla)
